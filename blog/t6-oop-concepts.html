Variable Scope

March 14, 2015

What is Variable Scope? Scope defines where in a program a variable is accessible. Ruby has 4 types of variable scope and one constant type variable:

  • Global Variable - $
  • Class Variable - @@
  • Instance Variable - @
  • Local Variable - [a-z] or _
  • Constant - [A-Z]

You can use the method .defined? to return the scope of the variable referenced. But it's alot easier to tell the scope of the variable for looking at the name.
